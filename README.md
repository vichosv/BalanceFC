⚽ BalancedFC — FUT de Barrio
App para armar equipos balanceados para el grupo de fútbol. Gestiona el plantel, edita las stats de cada jugador, genera equipos justos y guarda el historial de partidos.

¿Qué hace?

👥 Plantel — Agregás y eliminás jugadores, con posición preferida
📊 Stats — Editás 6 habilidades por jugador (Velocidad, Técnica, Defensa, Tiro, Stamina, Físico) con un radar visual
🃏 Cartas — Cada jugador tiene su carta estilo FUT con overall y tier
⚽ Partido — Seleccionás quién juega, generás equipos balanceados automáticamente o los armás a mano, con radar por equipo
🏆 Historial — Guardás resultados, MVP y estadísticas acumuladas de cada jugador

Cómo usar
Abrí index.html en cualquier navegador. No necesita internet ni instalación.
Para compartir con el grupo, publicalo en GitHub Pages (ver abajo).
Datos
Todo se guarda en el navegador (localStorage). Los datos no se pierden al cerrar, pero son locales al dispositivo.

Tecnologías
HTML + CSS + JS vanilla. Sin frameworks, sin dependencias, sin backend.
